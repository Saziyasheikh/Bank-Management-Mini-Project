##  🏦 Bank Management System (Python)

A simple *Bank Management System* built using Python. 

This mini project demonstrates how basic programming concepts like *conditional statements* and *user input handling* can be used to simulate real-world banking operations.



##  🚀 Features

- 💰 User can enter initial bank balance
  
- 📊 Check current balance
   
- ➕ Deposit money with validation
    
- ➖ Withdraw money with balance checking
   
- 🔄 Updated balance displayed after each transaction
   



##  🛠️ Technologies Used

- Python 🐍
   
- VS Code 💻  


##  🧠 Concepts Used

- Conditional Statements (if-elif-else)
    
- User Input Handling
  
- Basic Arithmetic Operations
  
- Decision-Making Logic  



##  📌 How It Works

1. User enters starting balance
     
2. Menu is displayed with options:
   
   - Check Balance
     
   - Deposit Money
       
   - Withdraw Money
     
   - Exit
     
3. Based on user choice:

   - Balance is displayed
      
   - Money is added or deducted
     
4. Updated balance is shown after each transaction
